## 爪哇

Java 是一种现代的，高级的，通用的，面向对象的编程语言。 该语言的设计目标是软件健壮性，耐用性和程序员生产率。 它可用于在 PC 或嵌入式系统上创建控制台应用程序，GUI 应用程序，Web 应用程序。
