## PyGTK

PyGTK 是 GTK+  GUI 库的一组 Python 包装器。 它提供了一套全面的图形元素和其他有用的编程工具，可用于创建桌面应用程序。 它是 GNOME 项目的一部分。 PyGTK 是免费软件，并根据 LGPL 许可。
