## Tkinter

_Tkinter_ 是与 Tk GUI 工具包的 Python 绑定。 Tk 是 Tcl 语言的原始 GUI 库。 Tkinter 被实现为 Python 封装程序，围绕着嵌入在 Python 解释器中的完整 Tcl 解释器。
