## wxPython

_wxPython_ 是用于创建桌面 GUI 应用程序的跨平台工具包。 使用 wxPython，开发人员可以在 Windows，Mac OS 和各种 Unix 系统上创建应用程序。 wxPython 是 wxWidgets 的包装，wxWidgets 是成熟的跨平台 C ++库。
